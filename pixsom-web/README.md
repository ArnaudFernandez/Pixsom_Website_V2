# Pixsom Website V2

Minimalist, modern landing page for Pixsom studio.

## Technologies
- **Next.js 14+**
- **Tailwind CSS v4**
- **Shadcn UI**
- **GSAP** (Animations)

## Local Development

To run the project locally and iterate:

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Start the development server**:
   ```bash
   npm run dev
   ```

3. **View the site**:
   Open [http://localhost:3000](http://localhost:3000) in your browser.

4. **Edit**:
   - `src/app/page.tsx`: Main structure.
   - `src/components/InteractiveBackground.tsx`: The background animation.
   - `src/components/Hero.tsx`: The text and animation logic.
   - `src/app/globals.css`: Global styles.

The page will hot-reload as you save changes.
